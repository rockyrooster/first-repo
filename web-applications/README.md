## Getting Started

### Option 1: Start from Scratch
1. Create a new folder with your project name
2. Add `index.html`, `style.css`, `script.js`
3. Open `index.html` in your browser to test

### Option 2: Use a Template
1. Copy the `starter-template/` folder
2. Rename it to your project name
3. Start editing the files

### Option 3: Use a Framework
1. Run `npm create vite@latest my-app` (for React/Vue)
2. Or `npx create-react-app my-app` (for React)
3. Follow the setup instructions

## Tips
- **Start simple**: Begin with just HTML and CSS
- **Add interactivity**: Learn JavaScript to make things dynamic
- **Use frameworks later**: Master the basics first
- **Test frequently**: Open your files in a browser often
- **Responsive design**: Make sure it works on mobile too
