# Web App Starter Template

A simple, beginner-friendly template to start building web applications.

## What's Included

- **index.html** - The structure of your webpage
- **style.css** - Styling to make it look good
- **script.js** - JavaScript for interactivity

## How to Use This Template

### Option 1: Quick Start
1. Copy this entire folder
2. Rename it to your project name
3. Open `index.html` in your browser
4. Start editing the files!

### Option 2: Live Server (Recommended)
1. Install VS Code
2. Install "Live Server" extension
3. Right-click `index.html` â†’ "Open with Live Server"
4. Changes auto-refresh in your browser!

## What Each File Does

### index.html
- The skeleton/structure of your website
- Contains all the text and content
- Links to CSS and JavaScript files

### style.css
- Makes your website look good
- Controls colors, fonts, layout, spacing
- Includes responsive design for mobile

### script.js
- Makes your website interactive
- Handles button clicks, form submissions
- Can fetch data, update content dynamically

## Customization Tips

### Change Colors
Edit `style.css` and change the color codes:
```css
background-color: #3498db;  /* Blue */
background-color: #e74c3c;  /* Red */
background-color: #2ecc71;  /* Green */
```

### Add More Pages
Create new HTML files and link to them:
```html
<a href="about.html">About</a>
```

### Add More Sections
Copy a `<section>` block in `index.html` and modify it.
