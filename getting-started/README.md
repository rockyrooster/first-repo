## ðŸ› ï¸ Essential Skills to Learn First

### 1. Command Line Basics
Learn to navigate using the terminal:
```bash
pwd          # where am I?
ls           # what's in this folder?
cd folder    # go into a folder
cd ..        # go back one level
mkdir name   # create a folder
```

### 2. Git Basics
Version control for your code:
```bash
git status              # what changed?
git add filename        # stage a file
git commit -m "message" # save changes
git push                # upload to GitHub
git pull                # download from GitHub
```

### 3. Text Editor / IDE
Choose one and learn it well:
- **VS Code** (recommended for beginners)
- Sublime Text
- Atom
- Or any IDE for your language

### 4. Reading Documentation
Learn to find answers:
- Google your questions
- Read official docs
- Check Stack Overflow
- Ask AI assistants (like me!)

## ðŸ“– Important Concepts to Understand

### Variables
Store data to use later
```javascript
let name = "Alex"
let age = 25
```

### Functions
Reusable blocks of code
```javascript
function greet(name) {
    return "Hello, " + name
}
```

### Conditionals
Make decisions in code
```javascript
if (age >= 18) {
    console.log("Adult")
} else {
    console.log("Minor")
}
```

### Loops
Repeat actions
```javascript
for (let i = 0; i < 5; i++) {
    console.log(i)
}
```
